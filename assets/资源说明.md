# 游戏资源文件夹

此文件夹用于存放游戏的自定义资源。

## 📁 文件夹结构

```
assets/
├── images/      # 图片资源
└── sounds/      # 音频资源
```

## 🎨 支持的资源文件

### 图片资源 (`images/`)

- **bird.png** - 小鸟图片（可选）
  - 推荐尺寸：30x30 像素或更大
  - 格式：PNG（支持透明背景）
  
- **background.png** - 背景图片（可选）
  - 推荐尺寸：800x600 像素
  - 格式：PNG 或 JPG

### 音频资源 (`sounds/`)

- **background.mp3**（或 `.ogg`, `.wav`, `.m4a`）- 背景音乐（可选）
  - 推荐时长：30秒-3分钟（自动循环）
  - 格式：MP3/OGG/WAV/M4A

## 📝 使用说明

1. 准备您的资源文件
2. 将文件放入对应的文件夹
3. 运行游戏，程序会自动检测并使用这些资源

**注意**：所有资源都是可选的，不提供文件也能正常运行游戏。

## 🎮 获取资源

### 免费素材网站
- 图片：[OpenGameArt.org](https://opengameart.org/) / [Kenney.nl](https://www.kenney.nl/)
- 音乐：[Incompetech.com](https://incompetech.com/) / [FreePD.com](https://freepd.com/)

### 在线工具
- 去背景：[Remove.bg](https://www.remove.bg/)
- 像素画：[Piskel](https://www.piskelapp.com/)
