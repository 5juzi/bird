# 🐦 飞翔的小鸟 

一个功能丰富的小游戏，使用Python和Pygame制作，支持自定义外部资源！

## ✨ 主要特性

- 🎮 **完整的游戏机制**：重力模拟、碰撞检测、背景滚动
- 🎨 **精美的视觉效果**：动画、粒子效果、生动的小鸟和云朵
- 🎵 **音效系统**：跳跃、得分、碰撞等音效，可调节音量
- 🎯 **难度系统**：简单、中等、困难三个难度级别
- 🔧 **自定义资源支持**：可导入外部图片和音乐文件
- 🏆 **道具系统**：护盾、慢动作、双倍分数
- 🖱️ **完整的UI**：鼠标和键盘双重操控
- ⚙️ **游戏设置**：音量调节、难度选择

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

或手动安装：

```bash
pip install pygame numpy
```

## 🎮 运行游戏

```bash
python flappy_bird.py
```

## 🕹️ 游戏操作

### 基本操作
- **空格键** / **鼠标左键**：让小鸟跳跃
- **ESC键**：暂停游戏 / 返回主菜单
- **方向键上下**：在菜单中选择选项
- **Enter键**：确认选择
- **鼠标**：点击菜单选项

### 游戏目标
- 让小鸟飞过管道障碍物
- 收集道具获得特殊能力
- 获得尽可能高的分数
- 避免碰到管道、地面或天花板

## 🎨 自定义资源

### 📁 文件夹结构

```
assets/
├── images/
│   ├── bird.png          # 小鸟图片（可选）
│   └── background.png    # 背景图片（可选）
└── sounds/
    └── background.mp3    # 背景音乐（可选）
```

### 添加自定义资源

#### 1. 自定义小鸟图片
- 准备一张PNG格式的小鸟图片（推荐30x30像素或更大）
- 支持透明背景
- 命名为 `bird.png`
- 放入 `assets/images/` 文件夹

#### 2. 自定义背景图片
- 准备一张800x600像素的背景图片
- 支持PNG或JPG格式
- 命名为 `background.png`
- 放入 `assets/images/` 文件夹

#### 3. 自定义背景音乐
- 准备音乐文件（支持MP3/OGG/WAV/M4A格式）
- 命名为 `background.mp3`（或其他支持的格式）
- 放入 `assets/sounds/` 文件夹

### 📖 详细说明

查看以下文件获取更多信息：
- `assets/资源说明.md` - 详细的资源使用指南，以及如何获取和制作游戏资源

**注意**：所有资源文件都是可选的！如果不提供，游戏会使用程序绘制/合成的默认效果。

## 🎯 难度说明

### 简单模式
- 管道速度较慢
- 管道间隙较大
- 无移动管道
- 无道具和额外障碍物

### 中等模式
- 平衡的速度和间隙
- 包含移动管道
- 包含道具系统
- 包含额外障碍物

### 困难模式
- 管道速度快
- 管道间隙小
- 包含移动管道
- 包含道具和障碍物

## 🐛 故障排除

**游戏无法启动**：
- 确保已安装所有依赖：`pip install pygame numpy`
- 检查Python版本是否 >= 3.7

**音效无法播放**：
- 确保numpy已正确安装
- 检查游戏内音量设置

**自定义资源未加载**：
- 检查文件名和路径是否正确
- 查看控制台输出的错误信息
